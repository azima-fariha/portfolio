# Azima Islam Fariha | Portfolio

A clean, CV-inspired portfolio website for Data Scientist & Analyst Azima Islam Fariha.

## View the site

1. **Open in browser:** Double-click `index.html` or right-click → Open With → your browser
2. **Local server (optional):** Run `npx serve .` or `python -m http.server 8000` in this folder for a smoother experience

## Structure

- **index.html** — Main page with all CV sections
- **styles.css** — Professional styling (Cormorant Garamond + DM Sans, teal accents)

## Sections

- Hero with contact info
- Professional Experience (Trakken, MVI PROPLANT, Munich RE, HUGO BOSS)
- Skills (ML, Programming, Platforms, Project Management, Collaboration)
- Education (M.Sc., B.Sc.)
- Certifications (Google, Agile/Scrum)
- Languages (Bengali, English, German)
